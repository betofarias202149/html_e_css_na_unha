Usando a propriedade "transform" com CSS


Neste exemplo, foi utilizada a propriedade "transform" e chamada a função "scale" dentro do CSS interno no documento, para assim, trabalhar com o tamanho do elemento.

Somente para dar uma cor e para base de treino, foi utilizada a propriedade "background" com "gradiente linear" de três cores com definição de 25 graus. Para maiores detalhes, veja diretamente no código.
